# Lab-1-LAW-Django

## How to run

1. run Gin first -> https://github.com/razaqa/Lab-1-LAW-Gin
2. ```python -m venv env```
3. ```env\Scripts\activate```
4. ```python manage.py runserver```

## How to access calculate

```bash
curl --location --request GET 'http://localhost:8000/calculate?a=7&b=8'
```
